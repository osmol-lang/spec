# The Osmol Specification

A declarative equilibrium language. **There is no send.**

- `osmol-spec-v0.1.md` — the language: five axioms, grammar, semantics, the seven forbidden forms.
- The founding documents: the fifty-year letter, the Osmosis Protocol, the engineering dissertation.
- The brand: two vessels, one level.

The specification — together with the conformance suite — defines the language.
Implementations serve it, never overrule it. Governed at [osmol-lang/rfcs](https://github.com/osmol-lang/rfcs). Licensed MIT.
